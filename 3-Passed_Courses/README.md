# VariousHomeworkAssignments
Various Homework and Assignments

Το πρόγραμμα της εργασίας Passed_Courses μοντελοποιεί ένα σύστημα ανάλυσης των περασμένων μαθημάτων ενός φοιτητή πανεπιστημίου.
Δίνεται έτοιμο γραφικό περιβάλλον (Main & ApplicationFrame) και ζητείται να αναπτύξετε κώδικα στις υπόλοιπες 2 κλάσεις (Course & Reader) καθώς και να κάνετε αλλαγές στο ApplicationFrame έτσι ώστε το σύστημα να λειτουργεί σωστά.
